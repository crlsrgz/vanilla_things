Insertion Sort

- Insertion sort will traverse the array using a for loop

- I will the search and compare whats the next lower value using a while lop

- If theres a lower value it will swapt it

- Open the console to view the steps with the swaps 

      
        function insertionSort(array) {
          const arrayLength = array.length;
          for (let i = 1; i < arrayLength; i++) {
            let j = i - 1;        
            while (j >= 0 && array[j + 1] < array[j]) {
              let temp = array[j + 1];
              array[j + 1] = array[j];
              array[j] = temp;
              j--;
            }
            
          }
          return array;
        }
